# 📊 ModuWork Sales Analysis

## 📌 Project Overview
This project analyzes ModuWork's transactional sales data using SQL to uncover 
actionable business insights around revenue performance, product trends, and 
customer behavior patterns.

**Goal:** Support data-driven decision making by transforming raw sales data 
into meaningful KPIs and business metrics.

---

## 🛠️ SQL Toolset Applied
This project showcases databases, utilizing SQL structures to extract actionable business insights:

* **CTEs & Subqueries:** Used to build modular, highly readable, and maintainable analytical pipelines.
* **Window Functions:** Applied for ranking algorithms (e.g., Top N products) and behavioral partitioning.
* **JOINS & UNIONS:** Leveraged to blend multi-table transactional schemas and consolidate historical data.
* **Conditional Logic (CASE Statements):** Implemented to dynamically classify customers and segment business metrics.
* **Aggregation Functions:** Used alongside grouping to compute revenue and performance KPIs.
* **DDL Basics & Data Types:** Structured with optimal data typing and schema definitions to ensure data integrity.
